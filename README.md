# Simple blog API with Laravel

Uma simples API de CMS desenvolvida com Laravel para um front-end com React.js.

## Informações

Desenvolvido com:

* PHP
* Laravel
* React.js

**Nota:** Desenvolvido e testado somente no Linux. Não testado no Windows.
